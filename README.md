# ポモドーロタイマー

シンプルなポモドーロタイマーアプリです。タスク管理機能付き。

## 機能

**タイマー**
- 作業・休憩タイマーの開始 / 一時停止 / リセット
- 作業時間終了後に休憩タイマーへ自動切り替え
- 作業時間・休憩時間を自由に設定可能（設定はブラウザに保存）
- 現在の状態（作業中 / 休憩中 / 一時停止）を表示

**タスク管理**
- タスクの追加・完了チェック・削除
- タスク一覧はブラウザに保存（リロードしても消えない）

## 使い方

`index.html` をブラウザで開くだけで動作します。ビルド不要です。

```
pomodoro-app/
├── index.html
├── app.js
└── style.css
```

## 技術スタック

- HTML / CSS / JavaScript（フレームワーク・ライブラリなし）
