# pomodoro-app

シンプルなポモドーロタイマーアプリです。

## 機能

- タイマーの開始・一時停止・リセット
- 作業時間終了後に休憩タイマーへ自動切り替え
- 作業時間・休憩時間を自由に設定可能
- 現在の状態（作業中 / 休憩中 / 一時停止）を表示

## 使い方

`frontend/index.html` をブラウザで開くだけで動作します。ビルド不要です。

```
pomodoro-app/
└── frontend/
    ├── index.html
    ├── app.js
    └── style.css
```
